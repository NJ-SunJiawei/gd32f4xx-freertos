# gd32f4xx-freertos
基于gd32f4xx移植freertos系统
1、支持freertos调度
2、支持fatfs+spi+dma+nor flash(gd25q40)
3、支持fatfs+sdio+dma+sd flash(内存卡32G)
4、支持uart printf
5、片上SRAM运行
6、boot+片外sdram运行
